# PersonnelManager


## To run locally:

Run the following commands:

`git clone https://github.com/EvonDong/PersonnelManager.git`

`npm install`

### To view the backend:

`cd backend`
`npm start`

go to: http://localhost:3000/

### To view the frontend:

`cd /attendencelist`
`npm install`
`npm run serve`

go to: http://localhost:8080/
